heapfy (the procedure to create a heap)

heap sort has 2 steps
 - create a heap 	--> O(n log n)
   insert elements from the heap root 
   (from left to the right in the array)
 - heapfy			--> O(n)
   each element in the last level is a heap without childrens 
   (from the right to the left)
   
 - delete all the heap keeping the deleted element at the 
   end of the array
